# Apps Script

1. Crie um projeto novo em [script.new](https://script.new).
2. Cole o conteudo de `Code.gs`.
3. Salve.
4. Opcionalmente crie a propriedade de script `APPS_SCRIPT_TOKEN`.
5. Implante como `Aplicativo da Web` com acesso `Qualquer pessoa com o link`.
6. Copie a URL `/exec` para `APPS_SCRIPT_WEB_APP_URL` no `.env.local`.
